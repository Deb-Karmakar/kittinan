# SAMANVAY Landing Page

**System for Agency Mapping And Nodal VAYavastha**

A modern, professional landing page for the SAMANVAY platform - a digital solution for streamlining PM-AJAY implementation across India through intelligent agency mapping and coordination.

## Features

- 🎨 Modern, professional design with Indian cultural elements
- 📱 Fully responsive and mobile-friendly
- 🌍 Three-tier stakeholder ecosystem showcase
- 🎯 Clear problem statement and solution presentation
- 🇮🇳 Indian flag colors and cultural design elements
- ⚡ Built with React and Tailwind CSS

## Stakeholder Roles

### 1. Central Ministry Admin (MoSJE – National Level)
- Smart GIS Mapping
- AI Alerts & Notifications
- Automated Communication Layer
- PFMS Integration
- National Timeline Tracking
- Reports & Analytics

### 2. State/UT Nodal Officer (Implementing Agency)
- State-level GIS Mapping
- Mobile-First PWA
- Project Timeline Tracker
- Issue Escalation Tools
- Agency Role Management

### 3. Executing Agency Admin (District/NGO/PWD/Education Dept.)
- Mobile-First Progress Updates
- Geo-tagged Photo Uploads
- Fund Utilization Logging
- Project Timeline Checklist
- Real-time Status Updates

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd samanvay-landing
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### Build for Production

```bash
npm run build
```

This builds the app for production to the `build` folder.
